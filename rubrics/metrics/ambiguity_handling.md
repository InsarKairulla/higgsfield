# Ambiguity Handling Rubric

- Determine whether the question is ambiguous based on the fetched evidence and the case context.
- Mark `pass` only if the answer explicitly disambiguates the relevant interpretations or clearly states the ambiguity before answering.
- Mark `fail` if the answer picks one interpretation without acknowledging the ambiguity when multiple plausible interpretations are present.
- Mark `mixed` if the answer hints at ambiguity but does not resolve it clearly enough for the user.
